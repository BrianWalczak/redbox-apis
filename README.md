# redbox-apis

Reimplementation of various redbox kiosk APIs

## Building

[pnpm](https://pnpm.io/installation) is required

-   Install dependencies: `pnpm install`
-   Build: `pnpm build`

## Usage

-   On the first start, run `pnpm migrate:prod`
-   `pnpm start`

## Notes
